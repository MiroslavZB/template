# template

A template for a Flutter project.

## Introduction

This project is a starting point for a Flutter application. It uses:

1. Getx 
   1. State management
   2. Navigation
   3. Dialogs
2. Firebase
   1. Authentication
   2. Firestore - Database
   3. Crashlytics - error loading
3. worldtime package for time formatting

## Utilities

This template features:
1. Fast way to add `Padding`
2. Fast way to add `Text`
3. Easy way to get a dialog without `BuildContext`
4. Simple Confirmation `Dialog`
5. Easy way to add `RichText`
6. Easy way to access null `String`s and `List`s
7. Horizontally scalable **Architecture**
8. Built-in `UserModel`
9. Built-in `Paging`
10. Built-in `Root`

## Getting Started
1. Create a new flutter app.
2. Fetch the files from this template.
3. Search for `// TODO` and make changes where needed.
4. Change the bundle id for ALL environments (Android, iOS, etc)
5. Run `flutter create .` in the terminal
6. Replace all instances of package imports (`package:template`) from the template app with your 
   app name.
7. Setup a Firebase project. 
8. Initialize Firebase.
